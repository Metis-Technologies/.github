# Support

How to get help with **Metis Technologies** projects.

## 💬 Questions and discussions

Use the **GitHub Discussions** of the relevant repository:

- [beautyflow](https://github.com/Metis-Technologies/beautyflow/discussions)
- [beautyflow-mobile](https://github.com/Metis-Technologies/beautyflow-mobile/discussions)
- [metis-website](https://github.com/Metis-Technologies/metis-website/discussions)

## 🐛 Bugs and feature requests

Open an **issue** in the repository you're interested in, using the provided issue templates.

## 🔒 Security issues

See [`SECURITY.md`](SECURITY.md) — please report security issues privately.

## 📩 Commercial inquiries

For partnerships, hiring, or commercial use of our open work, write to **hello@metis-technologies.com**.

## 🛠️ Response times

We are a small team (2 founders). Expect:

- **Issues**: triage within 5 business days
- **PRs**: first review within 7 business days
- **Security reports**: acknowledged within 72h
