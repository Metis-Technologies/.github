## 🧩 Cosa facciamo

### 🌐 Web app & piattaforme

Next.js + React + TypeScript con deploy su Cloudflare. Dal sito vetrina alle dashboard multi-tenant: responsive, accessibili, SEO-first e veloci.

### 📱 App mobile native

Flutter — iOS e Android da una sola codebase, UI native e fluide. Dal product discovery alla pubblicazione su App Store e Play Store.

### ⚙️ Backend & API

NestJS + Prisma + PostgreSQL. Isolamento multi-tenant by design, API-first documentate con OpenAPI.

### 🤖 Automazioni & AI

Integrazioni AI pragmatiche (non hype) e automazioni di processo: assistenti su WhatsApp, workflow tra sistemi, pipeline di dati. Meno lavoro manuale, più risultati.
